前言

Hexo 在安裝上實在是非常快速，用起來是輕鬆愉快，
相信你看完這篇文章也會覺得這也太容易了！

但深怕我這金魚腦會忘記，因此還是把它紀錄下來。

在這邊使用 Windows 的環境，如果需要知道其他系統的安裝方法可以參考 Hexo官方網站 。

設定 Repo 位置

在 Hexo 部落格檔案中找一個 _config.yml 檔案，打開後拉到最下面會看到一個 deploy: 標籤，
在這邊把相關的設定檔填上去。({user_name}是你的帳號)
