# 多言語ブログ投稿の例

ブログシステムの多言語機能を実証するサンプル投稿へようこそ！

## 特徴

このブログ投稿は複数の言語で利用できます：
- **English** (英語)
- **繁體中文** (繁体字中国語)
- **日本語** (このバージョン)

## 仕組み

多言語システムは以下を使用します：

1. **ファイル命名規則**：`slug.lang.md`
2. **自動言語検出** ファイル名から
3. **フォールバックメカニズム** デフォルト言語へ
4. **言語スイッチャー** コンポーネント

## コード例

```javascript
// 言語検出の例
function getPostLanguage(slug) {
  const parts = slug.split('.');
  return parts.length > 1 ? parts[parts.length - 1] : 'default';
}
```

## 利点

✅ より良いユーザー体験  
✅ SEOに優しい  
✅ ユーザーの言語設定を尊重  
✅ 簡単なコンテンツ管理
